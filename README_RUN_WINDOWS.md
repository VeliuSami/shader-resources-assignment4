# How to run (Windows / Visual Studio)

This repo contains a ready-to-open Visual Studio solution:

- `Assignment4Sami.sln`

## Steps

1. Install **Visual Studio 2022** with the **Desktop development with C++** workload (includes MFC).
2. Clone the repo.
3. Open `Assignment4Sami.sln`.
4. Select **x64** + **Debug**.
5. Press **F5**.

## Notes

- The working directory is set to the **solution directory** so paths like `ShaderWnd/vertex.glsl`, `models/bunny.obj`, `textures/right.jpg` load correctly.
